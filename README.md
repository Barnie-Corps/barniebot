### Prerequisites
- Node.js 16.x or higher
- TypeScript
- MySQL database

### Installation
1. Clone the repository
```bash
git clone https://github.com/santiadjmc/barniebot.git
cd barniebot
```

2. Install dependencies
```bash
npm install
```

3. Configure your environment variables
4. Build and run the bot
```bash
npm run build
npm start
```
